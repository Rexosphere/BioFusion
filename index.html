Method

Training Method

Phase 1

Frozen DenseNet121 backbone

Train classifier head during epochs 1-10 with learning rate 0.001.

Phase 2

Fine-tune denseblock4 + transition3

Unfreeze the final feature block during epochs 11-20 with learning rate 0.0001.

input

Forward pass

Image tensor flows through DenseNet121 and classifier head.

functions

Loss computation

Cross-entropy compares logits with Normal/Pneumonia labels.

schema

Backpropagation

Gradients are computed for trainable parameters.

sync

Optimizer update

Adam updates the classifier head or unfrozen layers by phase.

Preprocessing

  • Train split: 80%; validation split: 20%.
  • Resize to 224x224.
  • Grayscale converted to RGB for pretrained DenseNet121.
  • ImageNet normalization.
  • Minimal augmentation for clinically realistic variation.

Background

Research

DenseNet121

Dense connections support feature reuse and stable gradient flow for image classification.

CheXNet inspiration

The project follows the precedent of DenseNet-based chest X-ray pneumonia detection.

Kermany dataset

Pediatric chest X-rays from the Kermany pneumonia dataset define the classification task.

Grad-CAM explainability

Visual attention maps help reviewers inspect regions that influenced predictions.

Recall-focused threshold tuning

Thresholding emphasizes sensitivity for screening and triage use cases.

Robustness and quality

Degradation testing and data validation document model limits and reliability.
